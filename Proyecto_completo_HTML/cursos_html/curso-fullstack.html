Curso de Desarrollo Fullstack!

El curso de Desarrollo Fullstack está diseñado para proporcionar a los estudiantes una comprensión integral del desarrollo web tanto en el front-end como en el back-end. A lo largo de este programa intensivo de 16 semanas, los estudiantes aprenderán a crear aplicaciones web completas y funcionales utilizando las tecnologías más demandadas del mercado.

Duración y horario

Duración: 16 semanas

Modalidad: Online y presencial

Horario:

  • Lunes a Viernes: 6 PM - 9 PM
  • Sábados: 10 AM - 2 PM

Requisitos previos

Los participantes deben tener conocimientos básicos de HTML y CSS, así como familiaridad con al menos un lenguaje de programación.

Módulos del curso

  1. Introducción al Desarrollo Web

    • Historia y evolución del desarrollo web
    • Principios básicos de diseño web
    • Configuración del entorno de desarrollo

  2. Desarrollo Front-end

    • HTML5 y CSS3 avanzado
    • Frameworks y bibliotecas de CSS (Bootstrap, Tailwind)
    • JavaScript y ES6+
    • Frameworks de JavaScript (React, Vue.js)
    • Herramientas de construcción (Webpack, Babel)

Proyecto Final

Los estudiantes realizarán un proyecto final donde aplicarán todos los conocimientos adquiridos durante el curso.
